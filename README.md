# Projeto-reminDad
Sistema de lembrete de horários de refeição com confirmação do usuário.

Este é um projeto criado com o objetivo de servir como lembrete de horários de refeição. A princípio, foi pensado para o uso do meu pai que, após um traumatismo craniano, teve comprometimento da sua memória.
O nome Remind Dad remete "lembrar o pai" em inglês, e está sendo desenvolvido para resolver uma necessidade real: lembrar horários de refeição por meio de alarmes automáticos e que exijam interação do usuário para a confirmação da execução da tarefa.
